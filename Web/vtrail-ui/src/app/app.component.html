Prompts

The prompts will instruct our AI how to filter the data. The format is as follows
social network name + type of filter + the filter value
The supported social media platforms are
Tweeter
Reddit
Stack Overflow
The supported type of filters are
s - sentiment analysis (negative, neutral, positive)
c - categorization
Example:
redit+s+negative
